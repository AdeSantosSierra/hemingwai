# - La librería mega.py (1.0.8) y tenacity (<6.0.0) NO son compatibles con Python 3.13
#   debido a un problema con asyncio.coroutine (eliminado en Python 3.11+).
# - En lugar de mega.py, se recomienda usar MEGAcmd (instalado vía Snap) para interactuar con MEGA.
# - Instrucciones para instalar MEGAcmd:
#     sudo snap install mega-cmd
#     snap connect mega-cmd:removable-media
#     snap connect mega-cmd:home
#   Prueba: snap run mega-cmd.mega-version
#   Usa comandos como: snap run mega-cmd.mega-login email password
#
# Si necesitas usar mega.py con Python 3.10 o inferior:
# 1. Crea un entorno virtual con Python 3.10:
#      python3.10 -m venv .venv310
#      source .venv310/bin/activate
# 2. Instala mega.py==1.0.8 y tenacity==5.1.5.
# 3. Parchea mega.py manualmente (ver notas abajo).
#
# Notas sobre el parche para mega.py (solo si usas Python 3.11+ con mega.py):
# - Abre venv/lib/python3.13/site-packages/mega/mega.py
# - Comenta: from tenacity import retry, wait_exponential, retry_if_exception_type
# - Comenta o elimina el decorador @retry en la función _api_request
